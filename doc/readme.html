About FUMIPOS

Convert svg to canvas and to png

FUMIPOSのグラフはsvg形式で描かれており, そのままでは画像として保存できない. そのため一度svgを構成している要素をcanvas要素中に描画するということを行っている. 

ただし, ここではsvg要素のコンポーネントのスタイルを外部cssによって定義しているが, svg要素をcanvas要素に変換する際には外部cssは反映されない. 従って, svg要素内にstyle要素を追加したり, コンポーネントのstyle属性を追加することなどの対策が必要となる. 

ここでは, グラフの軸などのスタイルを定義した外部cssと, 凡例データから自動生成したデータスタイルcssの内容を, svg要素内のstyle要素にコピーすることで対応している. この時注意すべきことは，フォントサイズなど, サイズに関する値は必ず絶対値(px)で定義するということである. %, emなどを用いた定義では正しいスタイルとして反映されない. 

また, path要素はsvg要素内ではパス内部の塗りつぶし色がデフォルトで透明(fill: none;)だが, canvas要素内では黒色に塗りつぶされてしまう. 従って内部を塗りつぶさないpath要素は, style要素か属性にて明示的に"fill: none"を指定しなければならない.

既存の問題点として, circle要素のstrokeが反映されない場合がある. また, アンチエイリアスのために元のsvg画像に比べて若干ぼやけてしまう. 

Symbol highlighting

Normal plot モードでは，グラフ上のシンボル(circle, path)をマウスオーバーすると, 全てのグラフ上でデータが対応しているシンボルがハイライト表示(拡大, 不透明化など)されるようになっている. 

Symbol legend

シンボルの色については, ユーザーがあらかじめ定義したjsonファイルを読み込むことで一括して反映することができる.
